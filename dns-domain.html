DNS-Domain

Was ist eine Domain?

Eine Domain ist ein eindeutiger Name, der auf einen Computer oder eine Website auf dem Internet zeigt.

Was ist DNS?

DNS, kurz für Domain Name System, funktioniert wie ein Telefonbuch für das Internet.
Jede Website hat eine eigene IP-Adresse – eine Zahlenkombination, mit der Computer arbeiten.
Da sich Menschen aber schlecht Zahlen merken können, tippen wir stattdessen einen Domainnamen ein, zum Beispiel google.de.
Wenn du diesen Namen in den Browser eingibst, fragt dein Computer beim DNS nach: "Welche IP-Adresse gehört zu google.de?"
Das DNS schaut in seinem Telefonbuch nach, findet die passende IP-Adresse und schickt sie zurück. Erst dann kann sich dein Computer mit dem richtigen Server verbinden und die Website laden.
